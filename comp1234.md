## Week 1 - 
- Sumerian clay tablets (5kya), writing called Cuneiform - meaning wedge-shaped
- Proportional spacing; letters have different spacing
- Monospace; letters have their own space

## Week 2 - 
- 1 kilo = 1000/1024 (2^10), difference matters in some cases
- 0x to show number bases (0b binary, 0o octal, 0d decimal, 0x hexadecimal)
- Fixed vs Floating Point Notation (eg; in prices, decimal is fixed; 4.7988 --> 47988x10^-4)
- ASCII (American Standard Code for Info Interchange)
- Extended ASCII included 127 more characters

## Week 3 - 
- Markup languages; HTML, XML, YAML, MarkDown, Wik Text / Wiki Markup
- "Escaping" characters; \ used to separate specific characters from their assigned meaning

## Week 4 - 
- QWERTZ in Germany, E. Europe, Portugal
- AZERTY in French Canada
- Word Processors (MS Word) vs Text Editors (Notepad)
- Integrated Development Environment (IDE), language specific, eg; WebStorm for HTML, CSS, JavaScript; PyCharm for Python; Android Studio for Android apps
- Visual Studio Code; wildcards = representing any character, especially when using search/find function

## Week 5 - 
- Command Line Interface (CLI)
- Used in Terminal; getting to Terminal is different depending on OS
- Little computer resources needed, good for remote work over slow networks, some functionality has no GUI
